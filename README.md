# Desafio para vaga EQI


## Para rodar em sua maquina:
->Baixar
```javascript 
git clone #repo adress
```
-> Instalar dependencias

```cd frontend```
```javascript 
yarn && yarn start
```
```cd backend```
```javascript 
npm && nodemon index.js
```

>Não se esqueça de anexar as credencias ao banco que você estiver rodando!
